# Monorepo and Tooling

## Monorepo Strategy
- Single monorepo for applications, libraries, and infrastructure
- Clear separation between apps, shared libraries, and adapters

## Tooling
- Monorepo orchestrator: Nx
- Language: TypeScript
- Package manager: pnpm

Nx is used for:
- dependency graph awareness
- affected builds
- task orchestration
- consistent developer experience
